> 문제

아래 예제와 같이 개를 출력하시오.

> 출력

|\_/|  
|q p|   /}  
( 0 )"""\  
|"^"`    |  
||_/=\\__|  

> Tip

[escape 문자](https://arer.tistory.com/95)를  활용해서 출력할 수 있습니다.

> 풀이
```c
#include <stdio.h>
int main(void)
{
	printf("|\\_/|\n");
	printf("|q p|   /}\n");
	printf("( 0 )\"\"\"\\\n");
	printf("|\"^\"`    |\n");
	printf("||_/=\\\\__|");
	return 0;
}
```
