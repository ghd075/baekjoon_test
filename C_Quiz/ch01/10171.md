## 문제

아래 예제와 같이 고양이를 출력하시오.

## 출력

\\    /\\    
)  ( ')  
(  /  )  
\\(__)|    

## Tip

\를 표현하기 위해 두번 적어준다.

## 풀이
```c
#include <stdio.h>
int main(void)
{
	printf("\\    /\\\n");
	printf(" )  ( ')\n");
	printf("(  /  )\n");
	printf(" \\(__)|\n");
	return 0;
}
```
