## 문제

N을 입력받은 뒤, 구구단 N단을 출력하는 프로그램을 작성하시오. 출력 형식에 맞춰서 출력하면 된다.

## 입력

첫째 줄에 N이 주어진다. N은 1보다 크거나 같고, 9보다 작거나 같다.

## 출력

출력형식과 같게 `N*1`부터 `N*9`까지 출력한다.

## Tip

`for문`을 이용해서 출력을 합니다.

## 풀이
```c
#include<stdio.h>

int main()
{
	int i, num;

	scanf("%d", &num);

	for (i = 1; i < 10; i++) {
		printf("%d * %d = %d\n", num, i, num*i);
	}

	return 0;
}
```
